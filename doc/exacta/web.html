Beppo Web

Para acceder a Beppo Web debés utilizar el link de del menu Beppo. Luego de hacer click en el mismo, vas a entrar a la pantalla de Login, Ahi debés introducir tu "Nombre de Usuario" y "Contraseña", los cuales son provistos por Exacta Learning o algún punto de venta.

 

Página de Login de Beppo Web
 

 

Luego de entrar al sisterma, aparecerá la pantalla de Inicio de Beppo Web, la cuál se verá como la siguiente:

 

Pagina de Inicio de Beppo Web

 

El sistema es muy fácil de utilizar mediante su menu principal:

 

El menu de Beppo Web
 

 

A continuación una descripcion de los items del menu: 

  • Cerrar sesión: Cierra la sesion actual.
  • Inicio: La página de Inicio de Beppo Web.
  • Preferecias: Aca podés especificar tu uso horario y el lenguaje del sistema.
  • Precordinar una clase: Acá podés pre-cordinar una clase con un tutor. El procedimiento es muy fácil y consta de 3 pasos: elegir la materia, elegir un tutor y finalmente elegir un horario.
  • Contáctenos: Formulario para contactarnos.
  • Datos Personales: Tus datos personales.
  • Ayuda/Instrucciones. Esta página.

 

Además en la pagina de Inicio, se muestra información sobre las horas de uso que el usuario tiene en el sistema:

 

Informacion de las horas disponibles en Beppo
 

 

Exacta Learning no diferecia entre las horas de Acceso Instantaneo y horas de Clase Precordinadas. Por está razon, en tu cuenta siempre aparecerán las horas de  Acceso Instantaneo en 0. El tiempo se representa en horas, por lo tanto si el ususario dispone de una hora y media de uso del sistema, el mismo mostrará: "Horas de Clase Precordinadas disponibles: 1.5".

 

Además de la información de las horas que se disponen en el sistema, en la página de Inicio se pueden revisar TODAS las clases que han sido impartidas por los tutores y las aulas que se encuentran abiertas en el sistema.
